# GenAI Screen Reader

An AI-powered accessibility tool designed to help visually impaired users and fast readers consume web content efficiently.

The system extracts meaningful content from webpages, summarizes it using NLP techniques, and delivers audio output through text-to-speech.

## Key Features

- Intelligent content extraction from noisy webpages
- NLP-based summarization
- OCR support for image-based text
- Voice output using TTS
- Accessibility-first design

## Technologies

- Python
- Natural Language Processing (NLP)
- OCR
- Text-to-Speech (TTS)
- Web-based interfaces
