# Rental Platform (Privacy-first)

A concept web platform designed to connect property seekers and owners without exposing personal contact information.

The platform emphasizes privacy, role-based access, and secure in-app communication.

## Features

- Seeker / Owner entry flows
- Listings with address, images, and map pins
- No public phone/email exposure
- Encrypted in-app messaging (planned)
- Authentication-based access

## Architecture Plan

- Frontend: Static site (GitHub Pages)
- Backend (planned): Supabase or Firebase
- Maps integration

## Status

Concept and UI-level implementation.  
Shown as a design and architecture demonstration.
