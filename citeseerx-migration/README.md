# CiteSeerX Migration & Sitemap Automation

A large-scale archival project focused on migrating academic PDFs and generating automated indexes and sitemaps to improve discoverability.

The work emphasizes scalability, reliability, and reproducible workflows.

## Key Objectives

- Automated migration of large PDF collections
- Sitemap and index generation using checksums
- Improved archival discovery and organization

## Technologies

- Python
- Linux
- Automation scripting
- Elasticsearch (indexing concepts)

## Use Case

Designed for academic repositories and digital libraries handling large volumes of documents.

## Status

Completed as part of academic research and tooling exploration.
